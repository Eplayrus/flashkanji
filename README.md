# FlashKanji (Desktop)

Десктоп‑приложение для изучения кандзи: карточки, свайпы‑оценки (кнопками), SRS, SQLite, day streak и упражнение «вставь кандзи в текст».

## Что исправлено по новому ТЗ

- Приложение теперь **desktop**, а не web.
- **Day streak засчитывается только после просмотра 10 карточек** за день (просмотр = переворот карточки).
- При перевороте карточки на обратной стороне показывается:
  - чтение,
  - значение,
  - пример,
  - **перевод примера на русский**.
- Логотип: приложение ищет `assets/logo.png` (можно положить туда приложенное изображение). Если файла нет, показывается текстовый заголовок.

## Архитектура

- `flashkanji/db.py` — SQLite схема, миграции и сиды.
- `flashkanji/logic.py` — SRS, streak after 10 views, `LaterQueue`, упражнение.
- `flashkanji/desktop.py` — Tkinter UI и оркестрация пользовательского потока.
- `run.py` — точка входа.
- `tests_py/test_logic.py` — тесты ключевой логики.

## Таблицы SQLite

- `cards`
- `reviews`
- `profile`
- `learned_batches`
- `exercises`

## Запуск

```bash
python3 run.py
```

## Тесты

```bash
python3 -m unittest discover -s tests_py
```
