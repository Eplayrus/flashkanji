# FlashKanji (Kivy, mobile-first)

Мобильный интерфейс на **Kivy** c жестами свайпа, SRS, SQLite, day streak и мини‑тестом по выученным кандзи.

## Что реализовано

- UI на **Kivy** (под смартфон): крупная карточка, вертикальный layout, touch‑жесты.
- Жесты по карточке:
  - свайп вправо → помню,
  - свайп влево → не помню,
  - короткий тап → переворот карточки.
- При перевороте карточки показываются: чтения, значение, пример и перевод примера.
- Day streak засчитывается после просмотра 10 карточек за день (просмотр = flip).
- SRS с интервалами 1/3/7/14/30 дней.
- `laterQueue`: возврат «не помню» карточки позже (по количеству карточек или по минутам).
- После выучивания `LEARNED_BATCH_SIZE` кандзи автоматически появляется **небольшой тест**:
  - текст с пропусками `{K1}/{K2}/{K3}`,
  - нужно вставить изученные кандзи в слоты,
  - результат сохраняется в `exercises`, ошибки повышают `priority_boost`.
- SQLite-персистентность (`cards`, `reviews`, `profile`, `learned_batches`, `exercises`).

## Структура

- `flashkanji/mobile_app.py` — Kivy интерфейс + жесты + основной пользовательский поток.
- `flashkanji/db.py` — SQLite схема/миграции/сиды.
- `flashkanji/logic.py` — SRS, streak, очередь, проверка теста.
- `flashkanji/constants.py` — константы.
- `run.py` — запуск приложения.
- `tests_py/test_logic.py` — тесты логики.

## Запуск

1. Установить зависимости:

```bash
pip install kivy
```

2. (Опционально) положить лого в `assets/logo.png`.

3. Запустить:

```bash
python3 run.py
```

## Тесты

```bash
python3 -m unittest discover -s tests_py
```
