Хирагана и катакана

Hiragana

  • とうろく · touroku — регистрация
  • せいげん · seigen — ограничение
  • しょうにん · shounin — одобрение

Katakana

  • アカウント · akaunto — аккаунт
  • パスワード · pasuwaado — пароль
  • アクセス · akusesu — доступ

Слова-опоры

  • 登録 · touroku — регистрация. создание аккаунта
  • 制限 · seigen — ограничение. права доступа
  • 承認 · shounin — одобрение. модерация и заявки

Предложения с пропусками

  • アカウントを___します。
    アカウントを とうろくします。
    Я регистрирую аккаунт.
  • アクセスが___されています。
    アクセスが せいげんされています。
    Доступ ограничен.

Цели учебника

  • Разбирать формальные слова из уведомлений и правил.
  • Связывать кандзи с ситуацией: регистрация, подтверждение, ограничение, разрешение.
  • Ускорить узнавание редких форм через SRS и письмо.

Слова интерфейсов

На N2 много слов вроде 登録, 承認, 制限, 削除, 変更, 通知, 管理. Они нужны для реальных приложений, личных кабинетов и справочных страниц.

  • Учись видеть функцию слова: кнопка, предупреждение, статус или действие.
  • Читай примеры целиком, не вырывая кандзи из контекста.

Глубина памяти

Для N2 важна честность ответа. «Помню» должно означать: ты знаешь значение и можешь восстановить хотя бы одно чтение.

  • Если узнаёшь только по форме, но не читаешь, повтори раньше.
  • Используй избранное для трудных символов.
