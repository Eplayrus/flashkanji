## Важное замечание по авторским правам

Полные тексты с чужих сайтов, журналов, газет и учебников сюда не скопированы. Это было бы плохой идеей юридически и методически: часть материалов защищена авторским правом, а официальные JLPT-материалы прямо ограничивают перепечатку отдельных вопросов и произведений.

Поэтому файл устроен так:

- ссылки ведут на реальные источники, где можно читать оригиналы;
- учебные тексты ниже — новые, авторские, написанные под жанры "новости / журнал / рассказ / блог / экзаменационное чтение" и примерно под уровни JLPT;
- тексты можно свободно использовать для личного обучения, Flash Kanji, карточек, упражнений и разметки словаря.

# JLPT N5

## N5-一:日記 — あさのこと

**Жанр:** дневник / бытовой текст  
**Опора:** J-Nihongo, Tadoku, Bunpro N5  
**Цель:** простые глаголы, время, семья, еда, школа/работа

```jp
けさ、七時に起きました。
顔を洗って、水を飲みました。
朝ごはんはパンとたまごでした。
八時に家を出ました。
今日は天気がいいです。
駅まで歩きました。
電車の中で日本語の本を読みました。
少しむずかしかったですが、おもしろかったです。
```

### Проверочные вопросы

- この人は何時に起きましたか。
- 朝ごはんは何でしたか。
- 電車の中で何をしましたか。

## N5-二:会話 — かさ

**Жанр:** короткий диалог  
**Опора:** экзаменационные N5-диалоги, Japanesetest4you, Bunpro N5  
**Цель:** покупки, прилагательные, これ・それ・あれ

```jp
A:そのかさは新しいですか。
B:はい、きのう買いました。
A:きれいですね。高かったですか。
B:いいえ、あまり高くなかったです。
A:軽いですか。
B:はい、とても軽いです。でも、少し小さいです。
A:雨の日にいいですね。
B:はい。明日、雨ですから、使います。
```

### Проверочные вопросы

- Bさんはいつかさを買いましたか。
- かさは高かったですか。
- 明日の天気はどうですか。

## N5-三:お知らせ — 図書館

**Жанр:** объявление / notice  
**Опора:** официальные JLPT sample question types, Bunpro N5  
**Цель:** время, запреты, простая инструкция

```jp
図書館からのお知らせです。
月曜日は休みです。
火曜日から金曜日まで、朝九時から夕方六時まで開いています。
```

### Проверочные вопросы

- 図書館は月曜日に開いていますか。
- 土曜日は何時から何時までですか。
- 本はどのくらい借りることができますか。

**Жанр:** блог / путешествие  
**Опора:** Tadoku, Wasabi, ReadNihongo

```jp
桜はもう少しだけ残っていました。
写真を撮りたかったですが、人が多かったので、あまりいい写真は撮れませんでした。
それでも、春の京都はまた行きたいと思う場所でした。
```

# JLPT N3

## N3-一:生活記事 — 朝型の生活

**Жанр:** интернет-журнал / lifestyle  
**Опора:** NHK Easy, learner blogs, Bunpro N3  
**Цель:** ~ようになる、~ために、~わけではない、~によって

```jp
最近、「朝型の生活」を始める人が増えている。
朝早く起きて、仕事や学校へ行く前に勉強したり、運動したりする生活だ。
夜は疲れて何もできないという人にとって、朝の時間は静かで使いやすい。

しかし、朝型にすれば、だれでもすぐ元気になるわけではない。
人によって体のリズムは違うし、仕事の時間も違う。
夜遅くまで働く人が、無理に朝五時に起きると、かえって体をこわしてしまうこともある。

大切なのは、「朝型が正しい」と決めることではなく、自分の生活に合った時間を見つけることだ。
たとえば、毎日三十分だけ早く寝る、寝る前にスマートフォンを見ない、朝に一つだけ小さい予定を入れる。
このような小さな工夫なら、続けやすい。

生活を変えるときは、大きな目標より、続けられる習慣を作るほうが成功しやすい。
```

### Проверочные вопросы

- 朝型の生活とは、どんな生活ですか。
- どうして無理に朝早く起きるのはよくないことがありますか。
- 筆者は生活を変えるとき、何が大切だと言っていますか。

## N3-二:ニュース解説 — 外国人向けの案内

**Жанр:** новостное объяснение  
**Опора:** やさしい日本語 / NEWS WEB EASY style  
**Цель:** общественные услуги, ~ため、~ことになった、~だけでなく

```jp
市役所は、外国人向けの新しい生活案内サイトを作ることになった。
これまで、市の情報は日本語だけで書かれていることが多く、外国人の住民から「内容が分かりにくい」という声が出ていた。

新しいサイトでは、ごみの出し方、病院の探し方、災害のときに行く場所などを、やさしい日本語で説明する。
また、英語や中国語、ベトナム語などでも読めるようにする予定だ。

市役所の担当者は、「情報を多く出すだけではなく、本当に必要なときにすぐ分かる形にしたい」と話している。
そのため、スマートフォンでも見やすいデザインにし、地図やイラストも使うという。

外国人の住民が増える中で、言葉の壁を低くすることは、生活の安全にもつながる。
このサイトは来月から使える予定だ。
```

### Проверочные вопросы

- なぜ市役所は新しいサイトを作ることになりましたか。
- サイトではどんな情報を説明しますか。
- 担当者はどんな形の情報にしたいと言っていますか。

## N3-三:短編小説 — 雨の日の駅

**Жанр:** короткая новелла  
**Опора:** Tadoku higher levels, ReadNihongo N3, лёгкая современная проза  
**Цель:** описания, внутреннее состояние, ~ように見える、~ことにした

```jp
雨の日の駅は、いつもより少し暗く見える。
人々はかさを閉じながら、急いで改札を通っていく。
私はホームのベンチに座って、遅れている電車を待っていた。

となりに、小学生くらいの男の子が立っていた。
手には小さな花を持っている。
白い花だったが、雨で少しぬれて、重そうに下を向いていた。
男の子は何度も時計を見て、ため息をついた。

「だれかを待っているの?」と聞きたくなったが、知らない大人に話しかけられるのは怖いかもしれない。
だから、何も言わずにいた。

しばらくして、電車が来た。
そのとき、ホームの向こうから女の人が走ってきた。
男の子は急に明るい顔になり、花を高く上げた。
女の人は泣きそうな顔で笑った。

私はその二人を見て、少しだけ心が軽くなった。
雨の日にも、だれかを待つ時間は、ただ暗いだけではないのだと思った。
```

### Проверочные вопросы

- 男の子は何を持っていましたか。
- 男の子はだれを待っていたようですか。
- 筆者は最後に何を感じましたか。

# JLPT N2

## N2-一:雑誌コラム — 「便利さ」と疲れ

**Жанр:** журнальная колонка / essay  
**Опора:** N2 reading style, magazine essays  
**Цель:** абстрактное рассуждение, ~にすぎない、~とは限らない、~ざるを得ない

```jp
私たちの生活は、十年前に比べて驚くほど便利になった。
買い物は家から出なくてもできるし、分からない言葉はすぐに検索できる。
友人への連絡も、仕事の資料の共有も、数秒で終わる。
一見すると、時間の余裕は増えているはずだ。

ところが、実際には「前より楽になった」と感じている人ばかりではない。
むしろ、いつでも連絡が取れるために、仕事と休みの境目がなくなり、疲れを感じる人も増えている。
便利な道具は、本来、人間の負担を減らすためのものだ。
しかし、それを使う側の意識が変わらなければ、便利さは単なる速度の向上にすぎない。

たとえば、メールをすぐ返せるようになったからといって、すべてのメールにすぐ返事をしなければならないとは限らない。
情報を集めやすくなったからといって、すべてを知ろうとする必要もない。
道具が増えた社会では、何を使うかだけでなく、何を使わないかを選ぶ力が必要になる。

便利さに流されると、人はいつの間にか休むことに罪悪感を持つようになる。
だからこそ、現代の生活では、あえて通知を切る時間や、何もしない時間を作らざるを得ないのではないだろうか。
```

### Проверочные вопросы

- 筆者は、便利さが必ずしも楽さにつながらない理由をどう説明していますか。
- 「何を使わないかを選ぶ力」とは、どのような力ですか。
- 筆者は現代人にどんな時間が必要だと考えていますか。

## N2-二:社会記事 — 地方の空き家

**Жанр:** газетная статья / социальная тема  
**Опора:** newspaper-style N2 texts, NHK-style社会記事  
**Цель:** формальный текст, статистически звучащая подача без реальных данных, ~に伴い、~をめぐって、~とされる

```jp
地方の町では、空き家をどのように管理するかが大きな課題となっている。
人口の減少や高齢化に伴い、住む人がいなくなった家が増えているためだ。
空き家は、単に使われていない建物というだけではない。
長い間放置されると、屋根や壁が壊れ、台風や大雪のときに近所へ被害を出すおそれがある。

一方で、すべての空き家をすぐに壊せばよいというわけでもない。
古い家の中には、地域の歴史を伝える建物もある。
また、修理すれば、若い移住者の住まいや、小さな店、共同の仕事場として使える可能性もある。

最近では、自治体が空き家の所有者と利用したい人をつなぐ制度を作っている。
しかし、相続の手続きが複雑だったり、修理に多くのお金がかかったりするため、利用が進まない例も少なくない。

空き家をめぐる問題は、建物だけの問題ではない。
そこには、地域に人を呼び戻すこと、古いものをどう残すか、行政と住民がどう協力するかという、いくつもの課題が重なっている。
今後は、危険な建物を減らす対策と、使える建物を活かす工夫の両方が求められる。
```

### Проверочные вопросы

- 空き家が増えている主な理由は何ですか。
- 空き家をすべて壊せばよいわけではない理由は何ですか。
- 筆者は今後どのような対策が必要だと述べていますか。

## N2-三:現代小説風 — 古いカメラ

**Жанр:** современная проза / новелла  
**Опора:** Aozora-like literary reading, modern short fiction  
**Цель:** повествование, подтекст, устойчивые выражения, N2-ish文体

```jp
父の部屋を片づけていたら、押し入れの奥から古いカメラが出てきた。
黒い革のケースはところどころ白くなり、金属の部分には小さな傷がいくつも残っていた。
私はそのカメラを手に取ったが、使い方はまったく分からなかった。
今のカメラと違って、画面もなければ、撮った写真をすぐ確認することもできない。

机の引き出しには、同じカメラで撮られたらしい写真が何枚も入っていた。
若いころの母が笑っている写真、まだ小さかった私が公園で転んで泣いている写真、知らない海の前で父がまぶしそうに立っている写真。
どれも少しぼやけていて、色も薄くなっている。
それなのに、不思議と今のスマートフォンで撮った写真より、時間の重さがあるように感じられた。

父は、写真を撮るのが好きだったわけではない。
旅行に行っても、カメラを持ち歩くより、黙って景色を見ている人だった。
だから、残された写真が多くないことを、私は少し残念に思った。
もっと撮っておいてくれればよかったのに、と勝手なことを考えた。

しかし、写真が少ないからこそ、一枚一枚が強く残るのかもしれない。
何でも記録できる時代に、記録されなかった時間のほうが、かえって人の記憶に深く沈むこともある。
私はカメラをケースに戻し、捨てずに本棚の上へ置くことにした。
```

### Проверочные вопросы

- カメラはどこから出てきましたか。
- 筆者は古い写真にどんな印象を持ちましたか。
- 最後に筆者はカメラをどうしましたか。

# JLPT N1

## N1-一:評論 — 言葉を「やさしくする」こと

**Жанр:** критическая колонка / публицистика  
**Опора:** やさしい日本語, editorial-style N1 reading  
**Цель:** абстрактная аргументация, противопоставления, плотный синтаксис

```jp
「やさしい言葉」を使うことは、単に難しい語を避けることではない。
むしろ、何を伝え、何を削り、どの順序で相手に渡すかを設計する行為である。
専門家が専門用語を使わずに説明したつもりでも、前提となる知識を当然のものとして扱えば、その説明は結局、相手を置き去りにする。
言葉の難しさは、語彙そのものだけでなく、話し手が想定している世界の広さにも宿る。

近年、行政や報道の現場では、外国人住民や子ども、高齢者にも分かりやすい情報発信が求められている。
災害時の避難情報、医療制度、税金、学校からの連絡など、理解できなければ生活に直接影響する情報は多い。
その意味で、やさしい日本語は親切な補助ではなく、社会の安全装置の一部だと言える。

ただし、言葉をやさしくすることには危うさもある。
複雑な問題をあまりに単純化すれば、必要な背景や例外が失われる。
分かりやすさだけを優先した情報は、ときに誤解を招き、判断の材料を減らしてしまう。
したがって、やさしい言葉とは、内容を薄めるためのものではなく、理解へ至る入口を広げるためのものでなければならない。

本当に難しいのは、相手に合わせて言葉を低くすることではない。
相手の理解力を軽く見ず、それでも最初の一歩を踏み出せる形に情報を整えることである。
やさしさとは、情報を小さくする技術ではなく、相手が自分で考え続けられるように橋をかける技術なのだ。
```

### Проверочные вопросы

- 筆者は「やさしい言葉」をどのような行為だと述べていますか。
- やさしい日本語が「社会の安全装置」と言える理由は何ですか。
- 筆者は、言葉をやさしくするときの危うさをどう説明していますか。

## N1-二:新聞社説風 — 技術と責任

**Жанр:** газетная передовица / editorial  
**Опора:** newspaper/editorial style, N1 reading  
**Цель:** аргументация, сложные связи,社会・技術テーマ

```jp
新しい技術が社会に広がるとき、人々はしばしば「便利になるかどうか」という一点から評価しがちである。
しかし、技術の価値は便利さだけで測れるものではない。
それが誰の負担を減らし、誰に新たな負担を押しつけるのか。
どのような判断が自動化され、その結果に対して誰が責任を負うのか。
こうした問いを置き去りにしたまま導入を急げば、後になって社会全体が高い代償を払うことになりかねない。

たとえば、手続きの自動化は、行政や企業の効率を高める一方で、制度に慣れていない人を排除する危険を持つ。
画面の指示に従えば簡単だと言われても、その画面にたどり着けない人、読めない人、入力を間違えたときに助けを求められない人は存在する。
技術が中立であるかのように見えるのは、それを問題なく使える人の側から見ているからにすぎない。

もちろん、だからといって新しい技術を拒むべきだという話ではない。
問題は、導入の速さに比べて、失敗したときの修正や、取り残された人への支援があまりに遅いことにある。
技術は社会を変える力を持つが、その方向を決めるのは技術そのものではない。
設計する人、運用する組織、利用を促す制度、そして異議を唱える市民の存在である。

これからの社会に必要なのは、技術を恐れる態度でも、無条件に歓迎する態度でもない。
便利さの陰にある責任を見落とさず、導入後も問い続ける姿勢である。
技術を使う社会は、同時に、技術に使われないための知性を育てなければならない。
```

### Проверочные вопросы

- 筆者は、技術を評価するとき便利さ以外に何を見るべきだと言っていますか。
- 手続きの自動化にはどのような危険がありますか。
- 筆者が最後に述べる「技術に使われないための知性」とは、どのような姿勢ですか。

## N1-三:文学風短編 — 夜の橋

**Жанр:** литературная новелла  
**Опора:** Aozora Bunko / современная文学風読解  
**Цель:** образность, подтекст, сложная интерпретация

```jp
夜の橋を渡るたびに、私はいつも同じことを思い出す。
川面に並ぶ街灯の光は、風が吹くたびに細かく裂け、また何事もなかったようにつながる。
その動きを見ていると、人の記憶もこれと似たものではないかと思う。
一度途切れたように見えても、どこかでまだ続いている。
ただ、その続き方を、私たちは自分で確かめることができないだけなのだ。

十年前、この橋の上で友人と別れた。
別れたといっても、喧嘩をしたわけではない。
互いに忙しくなり、連絡の間隔が少しずつ伸び、やがて沈黙のほうが自然になった。
当時の私は、それを大人になることの一部だと考えていた。
人はそれぞれの場所へ行く。
それを止める権利など誰にもない、と。

けれども、権利がないことと、何も感じないことは同じではない。
失われた関係の多くは、決定的な出来事によって壊れるのではなく、返事を後回しにした数日、会う約束を曖昧にした一言、相手も分かっているはずだという甘えによって、静かに輪郭を失っていく。
そうして失ったものは、失った瞬間を指で示すことができないために、長い間、失ったことにさえ気づかれない。

橋の中央で立ち止まると、向こう岸のビルの窓に、まだ働いている人たちの白い光が点々と浮かんでいた。
私は携帯電話を取り出し、古い連絡先を探した。
名前はまだ残っていた。
消さなかったのか、消せなかったのか、自分でも分からない。

結局、私は何も送らなかった。
ただ、画面を閉じる前に、その名前をもう一度だけ見た。
連絡をしないこともまた、一つの選択なのだろう。
しかし、その選択に理由を与えられるほど、私はまだ大人になっていなかった。
```

### Проверочные вопросы

- 筆者は川面の光を見て、何と似ていると考えましたか。
- 友人との関係はどのように失われていきましたか。
- 最後に筆者が連絡をしなかったことは、どのような意味を持つと考えられますか。

- N5: короткие предложения, бытовые темы, минимум кандзи.

- N2: абстрактные темы, общественные статьи, сложные связки.
